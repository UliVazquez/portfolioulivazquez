Level Ro Confecciones

Para el proyecto final de un curso de desarrollo web, creé una página web responsive utilizando HTML, CSS y SASS. El objetivo del proyecto era desarrollar un e-commerce básico para promocionar el negocio de ropa de mi mamá. La página está diseñada para adaptarse a diferentes dispositivos, asegurando una experiencia de usuario óptima tanto en móviles como en computadoras. Utilicé HTML para estructurar el contenido y CSS para estilizar la página, implementando SASS para mejorar la eficiencia del código CSS. El sitio incluye una presentación visual atractiva de los productos, con imágenes de alta calidad y descripciones detalladas, además de una estructura de navegación sencilla e intuitiva, facilitando a los usuarios encontrar y explorar los productos. La página web sirve como una plataforma para aumentar la visibilidad del negocio de mi mamá, atrayendo potenciales clientes y facilitando la comunicación. Este proyecto no solo me permitió aplicar mis conocimientos técnicos en un contexto práctico, sino que también me dio la oportunidad de contribuir al crecimiento del negocio familiar.

Ecliptic Discovery

Para un hackatón de la NASA realizado en 2023, desarrollé una página web donde los usuarios pueden ingresar el nombre de un país y obtener información sobre los próximos eclipses visibles desde dicho país. Utilicé tecnologías web como HTML, CSS y JavaScript para crear una interfaz intuitiva y fácil de usar. La aplicación consulta una API de datos astronómicos para proporcionar información precisa y actualizada sobre los eclipses. Este proyecto no solo destacó por su utilidad y funcionalidad, sino también por su capacidad para fomentar el interés en eventos astronómicos y la ciencia espacial entre los usuarios.
